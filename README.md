# Coding Dojo's Potter Kata
      
## Information

https://codingdojo.org/kata/Potter/

## Installation

To install and run this project :

    gem install bundler

    git clone https://github.com/nad-abk/potter_katta.git
    cd potter_kata

    bundle install

## Usage

You can run the tests with:

    rspec --init
    rspec


Have fun doing the Kata!
